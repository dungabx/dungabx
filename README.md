### 📂 Featured Projects

#### 🌐 [AI-Driven SD-WAN Routing Optimizer](https://github.com/dungabx/sdwan-ai-optimizer)
> A smart network orchestration tool utilizing Python and Mininet to simulate and dynamically optimize routing paths in SD-WAN environments.
* **Key Features:** Employs reinforcement learning algorithms to adjust traffic flow dynamically, minimizing latency and maximizing bandwidth efficiency.
* **Tech Stack:** `Python` • `Mininet` • `SDN Controllers` • `Linux`

#### 🔒 [Encrypted P2P Voice & Video WebRTC Chat](https://github.com/dungabx/secure-p2p-webrtc)
> A serverless-signaling peer-to-peer secure communication platform using WebRTC, implementing robust cryptography at the application layer.
* **Key Features:** End-to-End Encryption (E2EE) utilizing **RSA-4096** for initial handshake and key exchange, and **ECC** (Elliptic Curve Cryptography) for lightweight digital signatures to prevent MitM attacks.
* **Tech Stack:** `WebRTC` • `JavaScript` • `CryptoAPI` • `Bootstrap`

#### 🏗️ [Automated Spine-Leaf Datacenter Fabric Simulation](https://github.com/dungabx/spine-leaf-fabric)
> High-performance infrastructure-as-code project to spin up a multi-tenant datacenter topology simulating VXLAN tunneling across a Spine-Leaf architecture.
* **Key Features:** Utilizes Docker containers to simulate switches and end-hosts, fully automated configuration deployment via Ansible, and traffic verification using custom Python script tests.
* **Tech Stack:** `Mininet` • `Docker` • `Python` • `VXLAN` • `Linux Routing`
